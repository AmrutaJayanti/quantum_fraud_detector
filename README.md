# 🧠 Quantum-Based Credit Card Fraud Detection System
A next-generation fraud detection system that combines **quantum-inspired machine learning**, **graph-based transaction analysis**, and **real-time monitoring** to identify suspicious credit card transactions with high accuracy and scalability.

## 🚀 Project Overview
Credit card fraud is increasingly complex and hard to detect using classical approaches alone. This project proposes a Quantum-Based Fraud Detection System that enhances traditional fraud detection using:

- ⚛️ Quantum-inspired feature encoding
- 📊 Machine Learning for risk scoring
- 🕸️ Graph-based transaction relationship analysis
- ⚡ Real-time fraud monitoring dashboard

The system visualizes transaction networks and flags fraudulent patterns dynamically.


## ✨ Key Features

- **🔍 Quantum-Inspired Fraud Detection Model**
  - Hybrid classical + quantum feature representation
  - Optimized for high-dimensional financial data

- **📈 Real-Time Transaction Monitoring**
  - Live updates via WebSockets
  - Streaming fraud alerts

- **🕸️ Transaction Relationship Graph**
  - Visualize card–device–merchant relationships
  - Highlight suspicious paths and fraud clusters

- **📂 CSV Batch Prediction**
  - Upload large transaction datasets
  - Perform batch fraud analysis

- **🧠 Risk Scoring Engine**
  - Each transaction assigned a fraud probability score
  - Enables priority-based investigation

- **🎯 Interactive Admin Dashboard**
  - View, edit, delete transactions
  - Graph visualization using ReactFlow
  - Real-time charts & analytics
